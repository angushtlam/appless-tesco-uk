# Appless Tesc* UK

This website allows you to access your UK Tesc* Clubcard without downloading its app. Enter your Clubcard number and the barcode will be available for scanning.

## Commands

```sh
npm install
npm run dev
npm run build
```

The app fills the viewport up to `430px`. On wider screens it remains centred
at that width, with the surrounding page visible behind it. Change
`--app-max-width` in `src/styles/global.css` to adjust the cap.
